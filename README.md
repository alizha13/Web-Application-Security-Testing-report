## Web Application Security Testing Project

This project demonstrates manual web application security testing
performed on an intentionally vulnerable web application.

### Key Focus
- Reflected Cross-Site Scripting (XSS)
- HTTP request and response analysis

### Tools Used
- Burp Suite Community Edition

### Methodology
- Manual interception and analysis of HTTP traffic
- Payload injection using Burp Suite Repeater

This project was conducted in a controlled and ethical testing environment.
